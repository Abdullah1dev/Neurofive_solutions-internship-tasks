# Customer Support Messages Dataset

This dataset contains 10 customer support messages that will be classified into one of the following categories:

- Complaint
- Question
- Praise

These messages will be used to compare the performance of Zero-Shot Prompting and Few-Shot Prompting across different Large Language Models (LLMs), including ChatGPT, Claude, and Gemini.
